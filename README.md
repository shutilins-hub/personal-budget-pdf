# Личный бюджет по PDF

Локальное MVP-приложение для ведения личного бюджета по PDF-выпискам банков. PDF обрабатываются на компьютере пользователя, внешние API и облачные сервисы не используются.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Запуск

```bash
streamlit run app.py
```

## Проверка

```bash
python3 -m compileall .
python3 -m unittest tests.test_pdf_parser
```

## Как пользоваться

1. Выберите профиль или создайте новый в боковой панели.
2. Загрузите PDF-выписку Сбера, Яндекс Банка или ВБ Банка.
3. Нажмите `Разобрать PDF`.
4. Выберите период отчёта и нажмите `Посчитать`.
5. Проверьте дашборд, таблицу операций и операции, которые требуют ручной разметки.

## Как работает классификация

Сначала применяются правила профиля пользователя из `data/profiles/<profile_id>/profile.json`. Затем применяются базовые правила из `config/default_profile.json`. Если правило не найдено, операция получает тип `Проверить` и попадает в ручную разметку.

Правила поддерживают поля:

- `contains_any`: достаточно одного совпадения в описании;
- `contains_all`: нужны все совпадения в описании;
- `direction`: `income` или `expense`;
- `amount_min`, `amount_max`: фильтр по модулю суммы;
- `bank`: банк операции;
- `type`: тип операции;
- `category`: категория бюджета;
- `personal_amount_mode`: `abs`, `0`, `-abs`, `bank_abs`.

Личные правила не зашиты в код. Они живут в JSON профиля.

## Что хранится локально

- SQLite-база `data/budget.sqlite3` с операциями.
- JSON-профили и правила в `data/profiles/`.
- Настройки категорий и базовых типов в `config/`.

PDF по умолчанию не сохраняются после обработки. В приложение добавлены функции маскирования телефонов, счетов, карт и ФИО в `privacy.py`.

## Важная модель суммы

`bank_amount` — сумма операции в банке.

`personal_amount` — сумма, которая реально влияет на личный бюджет.

Например, перевод самому себе может иметь `bank_amount = -5000`, но `personal_amount = 0`. Покупка на маркетплейсе может иметь `bank_amount = -1500`, но `personal_amount = 1500`.
