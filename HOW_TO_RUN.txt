Как запустить приложение

1. Распакуйте архив с папкой budget_app.
2. Откройте папку budget_app.
3. Дважды нажмите файл START_BUDGET_APP.command.
4. Если macOS спросит разрешение на запуск, разрешите запуск.
5. Дождитесь установки зависимостей и открытия браузера.

Адрес приложения:
http://localhost:8501

Если в окне Терминала всё равно появится вопрос Streamlit про Email:
- просто нажмите Enter один раз;
- после этого приложение продолжит запуск.

Если macOS пишет, что файл нельзя открыть:
1. Откройте Терминал.
2. Перетащите файл START_BUDGET_APP.command в окно Терминала.
3. Нажмите Enter.

Важно:
- PDF обрабатываются локально на компьютере.
- Исходные PDF не отправляются в облако.
- Для первого запуска нужен интернет, чтобы установить зависимости Python.
